# CPU-SCHEDULING_IMPLEMENTATION

## Installation

To run the project, follow these steps:


1. Navigate to the os_js folder where the project is located:
    ```sh
    cd os_js
    ```
2. Install the necessary dependencies:
    ```sh
    npm install
    ```

3. Install Vue Cli:
    ```sh
    npm install @vue-cli
    ```

4. Start the development server:
    ```sh
    npm run serve
    ```
